## Metodo de trabajo

### 1-Diagramas
En primer lugar, teniendo en mente el juego de casino que queremos realizar, vamos a hacer los diagramas de estado y pensar en que modulos/funciones vamos a necesitar.

### 2-Funciones
Una vez definidos todos los estados posibles y las funciones que vamos a necesitar, pasamos a describir que hace cada funcion a nivel logico para saber exactamente que vamos a crear y como funciona.

### 3-Codigo
Cuando ya tenemos toda la estructura logica montada, empezamos a crear el codigo de cada funcion. Para agilizar el proceso y ser mas eficientes, nos ayudaremos con IA. La IA no lo puede hacer todo, asi que nos vamos a asegurar de que nos corrija errores, ayude a escribir funciones y corrija el codigo para que tenga mas buenas practicas

### 4-Testing
En esta fase entramos cuando el juego teoricamente se encuentra terminado. En esta fase, comprobamos que todo funciona y nos aseguramos de que nada este fallando. Una vez pasadas las pruebas, pasamos a mergear el juego en la rama dev
<hr>

## Dev -> Main
Una vez tenemos todos los juegos preparados en dev, volvemos asegurarnos de que nada se ha roto y pasamos a mergear a main.
